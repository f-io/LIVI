# LIVI system package manifest, read by the installers and by the in-app check.
#
# Format: <section>|<package>|<probe>|<what it is needed for>
#
# <package> is the Debian name, used by the installers and for one-click install on apt hosts.
# <probe> tests the capability itself and works on any distro, so the check can still tell a
# Fedora or Arch user what is missing even though the package is named differently there.
#
# Probes:
#   cmd:<name>  the binary is on PATH, /usr/sbin or /sbin
#   py:<module> python3 can import the module
#   gst:<name>  gst-inspect-1.0 finds the element
#   file:<path> the path exists, one * standing in for a single directory name
#
# Sections:
#   core = required on every LIVI host
#   lite = additionally required on a host with no desktop session (Pi OS Lite, headless x86)
